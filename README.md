# BanhmiVim

Instant Neovim IDE just work (for me)

## Setup alias

### Linux
```zsh
alias banhmivim="<PATH>/BanhmiVim/banhmivim/banhmivim"
```

### Windows
```ps1
function banhmivim {
    & "D:\<PATH>\BanhmiVim\banhmivim\banhmivim.ps1" @args
}
```
